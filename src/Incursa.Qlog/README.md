# Incursa.Qlog

[`Incursa.Qlog`](../../README.md) is the packable library root for qlog-related models, abstractions, and tooling used for QUIC/qlog diagnostics and structured event work.

## Install

```bash
dotnet add package Incursa.Qlog
```

## Status

- This project is only a scaffold right now.
- No qlog serialization or event-mapping logic has been implemented yet.
- Future implementation should be driven by canonical requirements in `specs/requirements/qlog`.
